About The Project

Face Emotion Recognizer

Python / Flask

Deep Learning

HTML / CSS / JS

Docker / Jenkins

Github

An AI-Powered technology that analyses facial expressions from static images and videos in order to reveal information about the subject's emotional state.

A hybrid model that utilizes Haar Cascades object detector to detect faces in real-time, and a robust Deep Convolutional Neural Network (DCNN) to extract features from the image.

The application can detect one of the five emotions: Happy, Sad, Anger, Neutral and Surprise.
